Stock Marcket Data to Stock_prices table

Description: Used Pandas DataReader, Jupyter Notebook and Yahoo! Finance to create .csv files for each company in the health care industry on the Nasdaq. Used VBA scripts to throw all .csv files into one Excel Workbook with each stock being on its own tab. Each tab was labeled with their respective ticker symbol; therefore, we ran another VBA script to create a column in each worksheet with the ticker symbol. Ran one last VBA script to save each stock as its own .csv file and then ran a command prompt to merge all .csv files into one large .csv file.
